Express + AJAX Assignment
This is a backend project made with Express.js and Node.js.

Features
REST API endpoints for cats and users (GET, POST, PUT, DELETE)

Dummy data, no database

Upload cat images and generate a thumbnail (Sharp middleware)

AJAX tests with upload-test.html and REST Client (api-test.http)

Organised folders (controllers, models, routes, middlewares)

How to start
Install dependencies:
npm install

Run the server:
node src/index.js

Test endpoints using Postman, REST Client, or upload-test.html.
